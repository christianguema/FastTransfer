# FastTransfert
# FastTransfert est un projet django de transfere d'argent et de gestion de comptes virtuelle.
# Il permet de creer des comptes, de faire des transferts d'argent entre comptes et de visualiser les transactions.

# Installation
# 1. Clonez le repository
#    git clone
# 2. Installez les dependances
#    pip install -r requirements.txt
# 3. Appliquez les migrations
#    python manage.py migrate
# 4. Lancez le serveur de developpement
#    python manage.py runserver
# 5. Accedez a l'application via http://localhost:8000
